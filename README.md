# Mac Apps Search

Search and filter 1000+ Mac applications from [awesome-mac](https://github.com/jaywcjlove/awesome-mac) - without the ads.

**Live:** https://pxmyh.github.io/awesome-mac-apps/

## Features

- Fuzzy search by app name
- Filter by Free / Paid
- Links to source repos and App Store

## Development

```bash
bun install
bun run dev
```

## Update Data

```bash
bun run fetch-data
```
